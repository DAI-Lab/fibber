fibber.metrics.metric_utils.aggregate_metrics(dataset_name, paraphrase_strategy_name, experiment_name, results, customize_aggregation_fn_dict)[source]¶

Aggregate paraphrase metrics on a dataset to a summary and store in a dictionary.

Parameters

  • dataset_name (str) – the name of the dataset.

  • paraphrase_strategy_name (str) – the name of the paraphrase strategy.

  • experiment_name (str) – the name of the experiment.

  • results (dict) – the fibber dataset with paraphrases and metrics. The return value of compute_metrics.

  • customize_aggregation_fn_dict (dict) – A dict of customized aggregations. The dictionary is a mapping from aggregation name to aggregation function. The aggregation function should take one data_record, and returns a float.

Returns

the aggregated metrics.

(dict)
