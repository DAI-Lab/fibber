fibber.benchmark.customized_metric_aggregation.paraphrase_classification_accuracy_agg_fn(use_sim, ppl_score)[source]¶

This function makes a aggregation function for BERT classification metric.

The aggregation function outputs the after attack accuracy of the BERT classifier.

Parameters

  • use_sim (float) – the threshold for USESemanticSimilarity metric. The adversarial example should have the USE similarity higher than this threshold.

  • ppl_score (float) – the threshold for GPT2GrammarQuality metric. The adversarial example should have the perplexity ratio measured by GPT2 lower than this threshold.
