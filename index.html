These are the minimum commands needed to create a virtualenv using python3.6 for **fibber**: ```bash pip install virtualenv virtualenv -p $(which python3.6) fibber-venv ``` Afterwards, you have to execute this command to activate the virtualenv: ```bash source fibber-venv/bin/activate ``` Remember to execute it every time you start a new console to work on **fibber**!

Install from source¶

With your virtualenv activated, you can clone the repository and install it from source by running make install on the stable branch:

git clone git@github.com:DAI-Lab/fibber.git
cd fibber
git checkout stable
make install
